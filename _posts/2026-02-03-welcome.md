## How to add posts

Create markdown files in the `_posts` folder with this naming format:

```
YYYY-MM-DD-title-of-post.md
```

Each post needs front matter at the top:

```yaml
---
layout: post
title: "Your Post Title"
date: YYYY-MM-DD
---
```

Then write your content in markdown below.
